# ДСУ РЦСОО „Коле Нехтенин" - Штип

Веб страна за средно стручно училиште, изработена за натпревар по веб дизајн.

## Технологии

- **Backend**: PHP со BladeOne темплејти
- **Frontend**: Tailwind CSS, Tabler Icons
- **База**: MySQL
- **Хостинг**: InfinityFree

## Мултијазична поддршка

- Македонски (MK)
- English (EN)
- Türkçe (TR)
- Shqip (SQ)

## Структура

```
├── api/
│   └── lang/          # Language files (mk, en, tr, sq)
├── Art/               # Router, Blade engine
├── Public/            # Page templates (Blade)
├── assets/            # Images, CSS
├── index.php          # Entry point
├── sitemap.xml
└── robots.txt
```

## Страници

- Почетна
- За нас

- Новости
